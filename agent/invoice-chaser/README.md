# Invoice Chaser

![price: free](https://img.shields.io/badge/price-free-brightgreen) ![forkable](https://img.shields.io/badge/forkable-yes-blue) ![license: MIT](https://img.shields.io/badge/license-MIT-informational)

Politely chases overdue invoices and sends reminders.

## Fork this agent

- On Kavela: **[Open & fork →](https://kavela.ai/agents/invoice-chaser)**
- Or download [`agent.json`](./agent.json) and import it in your workspace (Studio → Import).

## What you'll connect

To run this agent you connect your own:

- billing_api

Secrets are never included in this artifact — you reconnect your own credentials.
